# Lista de Tarefas

Projeto simples de gerenciamento de tarefas desenvolvido com:

- HTML
- CSS
- JavaScript

## Funcionalidades

- Adicionar tarefas
- Marcar tarefas como concluídas
- Excluir tarefas
- Salvar tarefas no navegador (LocalStorage)

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## Objetivo

Projeto desenvolvido para praticar manipulação do DOM e lógica com JavaScript.
